A simple, user-friendly web application for parents to track their kids' chores, behavior goals, and points towards a custom rewards system.

## Features

- Multiple Child Profiles

- Real-time Point Calculation
- Rewards Redemption System

## Setup

This is a static React application deployed via GitHub Pages. Data is stored locally in your browser's Local Storage.
